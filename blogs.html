Question-01: What are the different ways to select an element in the DOM?

Answer: getElementByID(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll() etc.

Question-02: What is the difference between innerHTML, innerText, and textContent ?

Answer: innerHTML- gets HTML contents including HTML Tags. innerText- gets the visible text content without HTML Tags. textContent- gets all text content including hidden elements.

Question-03: What is event delegation in the DOM?

Answer: It allows to attach a single listener to parent element instead of adding multiple event listener to child elements.

Question-04: What is event bubbling in the DOM?

Answer: In this mechanism, events are propagated from the target element up to its ancestors.

Question-05:How do you create, add, and remove elements using JavaScript?

Answer: create- document.createElement(), add- appendChild(), remove- removeChild().
